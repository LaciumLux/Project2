# Phoenix Design - UI для верстки

Это статические HTML файлы всех UI компонентов проекта PhoenixMaui, вынесенные для удобной работы верстальщика.

## Структура папок

```
PhoenixDesign/
├── index.html              # Навигация по всем страницам
├── css/                    # CSS файлы
│   └── tailwind.phoenixui.min.css
├── Pages/                  # Страницы приложения
│   ├── exercise-page.html          # Главная страница упражнения
│   ├── game-page-scanning.html    # Сканирование QR-кода
│   ├── game-page-loading.html     # Загрузка мероприятия
│   ├── game-page-error.html       # Ошибка загрузки
│   ├── game-page-no-exercises.html # Нет упражнений
│   ├── game-page-loaded.html      # Мероприятие загружено
│   ├── settings-page.html         # Настройки
│   └── Persons/
│       └── person-list.html        # Список участников
├── Layout/                 # Компоненты layout
│   ├── top-bar.html       # Верхняя панель
│   └── tab-menu.html      # Нижнее меню
└── Dialogs/               # Диалоги
    └── person-dialog.html # Диалог участника
```

## Как использовать

1. Откройте `index.html` в браузере - это главная страница навигации
2. Переходите по ссылкам к нужным страницам
3. Все страницы используют Tailwind CSS через файл `css/tailwind.phoenixui.min.css`

## Описание страниц

### Pages

- **exercise-page.html** - Главная страница упражнения с мишенью и таблицей результатов
- **game-page-scanning.html** - Страница сканирования QR-кода для входа в мероприятие
- **game-page-loading.html** - Состояние загрузки мероприятия
- **game-page-error.html** - Состояние ошибки при загрузке
- **game-page-no-exercises.html** - Состояние когда в мероприятии нет упражнений Phoenix
- **game-page-loaded.html** - Страница мероприятия с выбором сквода, упражнения и участника
- **settings-page.html** - Страница настроек упражнения (выбор мишени, параметры серии)
- **Persons/person-list.html** - Список всех участников с возможностью добавления, редактирования, удаления

### Layout

- **top-bar.html** - Верхняя панель с выбором упражнения, участника и статусом BLE устройства
- **tab-menu.html** - Нижнее меню навигации с вкладками (Home, Persons, Settings, Game)

### Dialogs

- **person-dialog.html** - Модальное окно для добавления/редактирования участника

## Технологии

- **Tailwind CSS** - для стилизации (через DaisyUI)
- **HTML5** - разметка
- Все динамические данные заменены на статические примеры

## Примечания

- Все Blazor директивы (@page, @inject, @code и т.д.) удалены
- Динамические данные заменены на примеры
- Компоненты из SharedUi (PhoenixResultPanel, GameTopBar и т.д.) встроены как статический HTML
- SVG мишени показаны как placeholder - их нужно будет заменить на реальные SVG компоненты

## Для верстальщика

Вы можете:
- Изменять HTML структуру
- Менять классы Tailwind CSS
- Добавлять свои стили
- Тестировать изменения прямо в браузере

Все изменения будут видны сразу при открытии HTML файлов в браузере.
